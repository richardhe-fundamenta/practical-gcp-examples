5 · Concept & architecture

What happens on a request

sequenceDiagram
    actor U as User
    participant ADK as ADK agent (Cloud Run)
    participant MA as Managed agent (sandbox)
    participant GCS as Skills bucket
    U->>ADK: "prime factors of 360?"
    ADK->>MA: run_skill_task() · Interactions API
    MA->>GCS: read /.agent/skills (on demand)
    Note over MA: picks the prime-factors skill, runs Python in the sandbox
    MA-->>ADK: 360 = 2^3 x 3^2 x 5
    ADK-->>U: answer + behind-the-scenes
