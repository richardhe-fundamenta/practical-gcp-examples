# Output contract

One question -> one visual. Never a multi-panel dashboard.

- **Chart type:** line for trend-over-time, bars for category comparison.
- **Title = the finding**, phrased as a conclusion, not a label.
- **Deltas shown**, not just levels (annotation or reference line).
- **Compact table** of exact numbers beneath the chart.
- **One "so what"** line.
- **Default format = PNG** (single image). Interactive HTML is opt-in and only
  on a harness-confirmed trusted surface.

If multiple findings exist, choose the most decision-relevant one. Adding panels
is a contract violation.
